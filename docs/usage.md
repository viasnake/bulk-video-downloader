# Usage

## 基本操作

1. URL をテキストエリアに1行ずつ入力します。
2. 「リストに追加」を押します。
3. 保存先・追加オプション・並列数を設定します。
4. 「開始」を押してダウンロードします。
5. 停止する場合は「停止」を押します。

## URL ファイルの読み込み

- 「ファイル読込」から URL 一覧のテキストファイルを指定します。
- 空行は無視されます。

## URL の削除

- 一覧から対象行を選択し「選択削除」を押します。
- ダウンロード中は削除できません。

## 追加オプション例

- 音声のみ: `-x --audio-format mp3`
- 最高画質: `-f best`

## 進捗と状態

- 状態: 待機 / 実行中 / 完了 / エラー
- 進捗: URL 単位で%表示
